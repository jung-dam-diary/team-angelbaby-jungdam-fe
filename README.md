## 🚆 Server Start
1. `.env` 파일 작성 후, `REACT_APP_HOME, REACT_APP_API_URL, REACT_APP_HOME_URL, REACT_APP_OAUTH2_REDIRECT_URL` 환경 변수 설정 
2. `yarn install`
3. `yarn start`
